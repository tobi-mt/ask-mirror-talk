# Ask Mirror Talk Campaign Launch Pack

This pack is for the first 3 acquisition themes:

- `grief`
- `loneliness`
- `purpose`

Use these links and copy after deploying the latest child theme so campaign attribution and product events are captured.

## Core Goal

Drive qualified first asks, not just clicks.

Primary conversion:

- `campaign_landing_viewed -> question_submitted`

Secondary conversion:

- `question_submitted -> share_cta_used`
- `question_submitted -> saved insight`
- day-2 return

## What To Watch In Analytics

After deployment and traffic:

1. Check whether `campaign_landing_viewed` starts appearing.
2. Check whether `question_submitted` contains campaign metadata.
3. Compare conversion by:
   - `utm_campaign`
   - `utm_source`
   - `campaign theme`
4. Watch which campaign produces the best:
   - first ask rate
   - second ask rate
   - share CTA usage

## Success Thresholds For Week 1

Healthy early signal:

- `20%+` of campaign landings become a submitted question
- one theme clearly outperforms the others
- invite/share events begin appearing after first answers

Weak signal:

- lots of landings, few submitted questions
- one campaign gets clicks but poor first-ask conversion
- users land but do not interact with the prefilled prompt

If that happens, tighten the opening copy and shorten the first prompt.
